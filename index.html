Fitur Pembelajaran

Beberapa fitur pembelajaran yang digunakan, yaitu sebagai berikut:

HTML

HTML (Hyper Text Markup Language) adalah bahasa markup yang digunakan untuk membuat Halaman web.HTML terdiri dari kombinasi teks dan simbol yang disimpan dalam sebuah file.

CSS

CSS (Cascading Style Sheet) adalah bahasa yang digunakan untuk mengatur tampilan (gaya) elemen-elemen HTML. Dengan CSS, kita dapat memberikan warna background, border, mengatur posisi elemen, ukuran font dan lain sebagainya.

Javascript

Javascript adalah bahasa pemrograman (berjalan pada web browser) yang digunakan untuk membuat halaman web yang lebih interaktif. Dengan javascript, kita bisa membuat banyak fitur pada halaman web.
